# Jiayin Masterclass — Rebuild Demo

Redesign concept for [Jiayin Masterclass](https://www.jiayinmasterclass.dance/), an academy of classical ballet in Melbourne — Vaganova method, RAD examinations, faculty formed at The Australian Ballet.

## What's here

| Path | Contents |
|---|---|
| `teaser/` | Teaser homepage concept — photography-led, self-contained (brand fonts embedded, no build step) |
| `Jiayin-Rebuild-Blueprint.pdf` | Full rebuild blueprint: brand direction, website architecture, operations platform design, tech stack, roadmap |
| `Jiayin-Proposal.pdf` | Client proposal: three engagement paths and a modular pricing menu |

## Run the demo

```
node teaser/server.js
```

Then open http://localhost:4173 — or simply open `teaser/index.html` in a browser.

## Notes

- Photography is drawn from the school's existing site for concept purposes; usage rights and photographer credit (Simon C) to be confirmed before any public launch.
- Brand system: Blush Ivory / Stage Noir / Pointe Rose / Antique Gold / Merlot; Bodoni Moda display over Hanken Grotesk text.
